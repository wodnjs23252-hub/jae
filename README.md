# 프롬프트 빌더

간단한 목적만 입력하면 바로 쓸 수 있는 AI 프롬프트를 자동 생성하는 정적 웹앱입니다.

## 실행

정적 파일만으로 동작하므로 `index.html`을 브라우저에서 열어도 됩니다.

로컬 서버로 확인하려면 아래 명령을 실행합니다.

```bash
node server.mjs
```

기본 주소는 `http://127.0.0.1:4173`입니다.

## 주요 기능

- 목적 한 줄 입력만으로 프롬프트 자동 생성
- 이메일, 요약, 보고서, 마케팅, 개발 요청, 학습 계획 자동 분류
- Role, Context, Task, Constraints, Output Format 구조 자동 구성
- 선택 입력은 접힌 세부 옵션으로 제공
- 복사한 프롬프트를 브라우저 로컬 저장소에 저장
