# 프롬프트 빌더

AI 초보자와 직장인을 위한 인터랙티브 프롬프트 자동 생성 MVP입니다.

## 실행

정적 웹앱이므로 `index.html`을 브라우저에서 열면 바로 사용할 수 있습니다.

## 주요 기능

- 목적 입력 기반 템플릿 추천
- 이메일, 요약, 보고서, 마케팅, 개발 요청, 학습 계획 템플릿
- Role, Context, Task, Constraints, Output Format 구조의 프롬프트 자동 생성
- 품질 점수와 보완 안내
- 클립보드 복사
- 최근 생성 프롬프트 로컬 저장

## 배포

`main` 브랜치에 push되면 GitHub Actions가 GitHub Pages로 정적 사이트를 배포합니다.
